# How to add your Colab notebooks here

1. In Google Colab, go to  File → Download → Download .ipynb
2. Rename the files:
   - Phase 1 notebook  →  Phase01_MCTNet.ipynb
   - Phase 3 notebook  →  Phase03_CASoilV2.ipynb
3. Drop them in this  notebooks/  folder before pushing to GitHub
